Detailed Description

The ToolBar class represents a graphical user interface toolbar.

A class that represents a toolbar.

This class inherits from the AGUI class.

The ToolBar class is a subclass of AGUI and provides a way to display and interact with a toolbar. A toolbar is a graphical control element that contains buttons or icons that allow the user to perform actions.
