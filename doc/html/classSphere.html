Detailed Description

Represents a sphere primitive in a ray tracer.

The Sphere class inherits from the APrimitive class and provides functionality to represent a sphere in a ray tracer scene. It stores the center and radius of the sphere and implements the hit() function to check for ray-sphere intersection.
