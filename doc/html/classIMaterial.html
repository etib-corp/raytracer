Detailed Description

The IMaterial class represents an interface for materials used in ray tracing.

This class provides a common interface for different types of materials that can be used in ray tracing. It defines a pure virtual function scatter() that calculates the scattered ray and attenuation color based on the incoming ray and hit record.

Calculates the scattered ray and attenuation color for a given incoming ray and hit record.

This pure virtual function is implemented by derived classes to calculate the scattered ray and attenuation color based on the incoming ray and hit record. The function returns true if scattering occurs, and false otherwise.
