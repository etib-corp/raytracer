Detailed Description

Builder class for creating Lights objects.

The LightsBuilder class provides a convenient way to construct Lights objects with different properties and configurations. It follows the Builder design pattern and allows the user to set ambient and diffuse values, add point lights, and set the type of light (directional or not). The build() method is used to obtain the final Lights object.
