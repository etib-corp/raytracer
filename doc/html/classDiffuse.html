Detailed Description

The Diffuse class represents a material with diffuse properties.

The Diffuse class is a subclass of AMaterial and represents a diffuse material. A diffuse material is a material that scatters light in all directions.

This class inherits from the AMaterial class and provides an implementation for the emitted() function, which calculates the emitted color of the material based on the given ray, hit record, texture coordinates, and point in 3D space.
