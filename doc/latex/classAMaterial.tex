\doxysubsection{Detailed Description}
Abstract base class for materials used in ray tracing. 

The \doxylink{classAMaterial}{AMaterial} class provides a common interface for materials used in ray tracing. It inherits from the \doxylink{classIMaterial}{IMaterial} interface and defines a virtual function for scattering rays. It also contains a protected member variable for the material\textquotesingle{}s albedo color.
