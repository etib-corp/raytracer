\doxysubsection{Detailed Description}
A class for parsing configuration files and extracting camera and primitive information. 

The \doxylink{classCFGParser}{CFGParser} class is responsible for reading a configuration file and extracting various camera and primitive properties. It provides methods to retrieve camera resolution, position, rotation, field of view, and other camera settings. It also provides methods to retrieve information about spheres, cylinders, lights, and other primitives defined in the configuration file.
