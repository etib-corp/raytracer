\doxysubsection{Detailed Description}
The \doxylink{classGUI_1_1ToolBar}{Tool\+Bar} class represents a graphical user interface toolbar. 

A class that represents a toolbar.

This class inherits from the \doxylink{classGUI_1_1AGUI}{AGUI} class.

The \doxylink{classGUI_1_1ToolBar}{Tool\+Bar} class is a subclass of \doxylink{classGUI_1_1AGUI}{AGUI} and provides a way to display and interact with a toolbar. A toolbar is a graphical control element that contains buttons or icons that allow the user to perform actions.
