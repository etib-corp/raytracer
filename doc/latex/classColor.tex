\doxysubsection{Detailed Description}
The \doxylink{classColor}{Color} class represents a color with red, green, blue, and alpha components. 

The \doxylink{classColor}{Color} class has a constructor that takes red, green, and blue values as parameters, along with an optional alpha value. If the alpha value is not provided, it defaults to 255.

The class also has a copy constructor that creates a new \doxylink{classColor}{Color} object from an existing one.

The \doxylink{classColor}{Color} class has private data members for the red, green, blue, and alpha components.
