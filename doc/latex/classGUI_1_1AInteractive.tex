\doxysubsection{Detailed Description}
Abstract base class for interactive GUI elements. 

This class provides a basic interface for interactive elements, including a virtual \doxylink{classGUI_1_1AInteractive_a35d969327a1f0cf3898cd6189cff4923}{display()} method that must be implemented by derived classes.
