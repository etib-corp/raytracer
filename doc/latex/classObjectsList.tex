\doxysubsection{Detailed Description}
The \doxylink{classObjectsList}{Objects\+List} class represents a list of objects in a scene. 

This class inherits from the \doxylink{classAPrimitive}{APrimitive} class and provides functionality to store and manage a collection of objects. It allows adding objects to the list, clearing the list, and checking for intersection with a ray.
