\doxysubsection{Detailed Description}
The \doxylink{classIMaterial}{IMaterial} class represents an interface for materials used in ray tracing. 

This class provides a common interface for different types of materials that can be used in ray tracing. It defines a pure virtual function \doxylink{classIMaterial_aa41840b65d4da0af8bcdd353230ce753}{scatter()} that calculates the scattered ray and attenuation color based on the incoming ray and hit record.
