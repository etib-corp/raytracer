\doxysubsection{Detailed Description}
Represents a rectangle shape. 

The \doxylink{classMath_1_1Rectangle}{Rectangle} class provides functionality to create and manipulate rectangle objects. It includes methods to calculate the area, perimeter, and check if a point is inside the rectangle.
