\doxysubsection{Detailed Description}
The \doxylink{classLambertian}{Lambertian} class represents a \doxylink{classLambertian}{Lambertian} material in a ray tracer. 

\doxylink{classLambertian}{Lambertian} materials are diffuse materials that scatter light in all directions. They have a constant albedo, which determines the color of the material.
