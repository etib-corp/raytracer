\doxysubsection{Detailed Description}
The \doxylink{classITexture}{ITexture} class represents an interface for textures in a raytracer. 

This class provides a common interface for different types of textures that can be used in a raytracer. It defines a pure virtual function {\ttfamily value} that returns the color of the texture at a given UV coordinate and point.
