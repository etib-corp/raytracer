\doxysubsection{Detailed Description}
Represents a checkerboard texture. 

The \doxylink{classChecker}{Checker} class is a concrete implementation of the \doxylink{classATexture}{ATexture} abstract class. It represents a checkerboard texture that alternates between two given textures or colors.
