\doxysubsection{Detailed Description}
Represents a directional light source in a raytracer. 

The \doxylink{classDirectionnal}{Directionnal} class is a subclass of \doxylink{classAMaterial}{AMaterial} and represents a directional light source in a raytracer. It can be initialized with a texture or a color.
