\doxysubsection{Detailed Description}
The \doxylink{classGUI_1_1Button}{Button} class is a subclass of \doxylink{classGUI_1_1AInteractive}{AInteractive} and represents a graphical button in the GUI. 

The \doxylink{classGUI_1_1Button}{Button} class has a constructor that takes a title and a function pointer as parameters. The function pointer is used to define the action that will be performed when the button is clicked. The \doxylink{classGUI_1_1Button_a47a6c68263417c089454460dca8bf5bd}{display()} method is used to render the button on the screen.
