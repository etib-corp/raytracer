\doxysubsection{Detailed Description}
Represents a cylinder primitive in a raytracer. 

The \doxylink{classCylindre}{Cylindre} class inherits from the \doxylink{classAPrimitive}{APrimitive} class and represents a cylinder primitive in a raytracer. It defines the properties and behavior of a cylinder, such as its position, depth, and radius.
