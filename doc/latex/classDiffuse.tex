\doxysubsection{Detailed Description}
The \doxylink{classDiffuse}{Diffuse} class represents a material with diffuse properties. 

The \doxylink{classDiffuse}{Diffuse} class is a subclass of \doxylink{classAMaterial}{AMaterial} and represents a diffuse material. A diffuse material is a material that scatters light in all directions.

This class inherits from the \doxylink{classAMaterial}{AMaterial} class and provides an implementation for the \doxylink{classDiffuse_aef3ba3c8c6ee9d3de0b3b07f9043d490}{emitted()} function, which calculates the emitted color of the material based on the given ray, hit record, texture coordinates, and point in 3D space.
