\doxysubsection{Detailed Description}
Represents a sphere primitive in a ray tracer. 

The \doxylink{classSphere}{Sphere} class inherits from the \doxylink{classAPrimitive}{APrimitive} class and provides functionality to represent a sphere in a ray tracer scene. It stores the center and radius of the sphere and implements the \doxylink{classSphere_a2bff644306d2f43bc9cf7c30ec908de1}{hit()} function to check for ray-\/sphere intersection.
