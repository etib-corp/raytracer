\doxysubsection{Detailed Description}
Builder class for creating \doxylink{classLights}{Lights} objects. 

The \doxylink{classLightsBuilder}{Lights\+Builder} class provides a convenient way to construct \doxylink{classLights}{Lights} objects with different properties and configurations. It follows the Builder design pattern and allows the user to set ambient and diffuse values, add point lights, and set the type of light (directional or not). The \doxylink{classLightsBuilder_a2216eac8fa92104e19b187b8b8b30d23}{build()} method is used to obtain the final \doxylink{classLights}{Lights} object.
